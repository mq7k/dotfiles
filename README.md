### Collections of my configuration file (dotfiles) on Hyprland Arch Linux.

#### How to use
1. Clone this repo.
2. `cd dotfiles` (Required, do NOT run the script from outside the directory).
3. Run `script.sh`.
4. Done.

#### Notes
Designed for a clean Arch Linux install using pacman with default configuration.
No support for derivatives or partially configured systems.

#### Requirements
- Active internet connection.
- Run **inside** the repository directory.
- Execute as a normal user.

#### Warning
Running this on a non-fresh system may break the current environment.
I do not take any responsability if this script will break your system.
